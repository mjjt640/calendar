# Calendar

Android Java calendar app scaffold with a single `app` module, MVVM-oriented package layout, and starter documents for project planning.

## Current contents

- Android project skeleton under `app/`
- Home screen starter UI with `MainActivity` and `HomeFragment`
- Initial `HomeViewModel` test slice
- Project reference documents:
  - `README_calendar_app.md`
  - `SRS_calendar_app.md`

## Next steps

1. Open the project in Android Studio.
2. Let Android Studio sync Gradle dependencies.
3. Install SDK 34 or 35 if prompted.
4. Run the app on an emulator or Android device.
