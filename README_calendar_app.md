# 日程表 App

## 1. 项目简介

日程表 App 是一款基于 Android 原生 Java 开发的手机端日程管理应用，面向个人日常计划、学习安排、工作待办和重要事项提醒等场景。项目以本地离线可用为核心，支持日程创建、编辑、删除、查看、重复规则、提醒通知、数据导入导出、主题切换及手势交互优化。

本项目采用分层设计与规范化工程结构，便于课程设计、毕业设计、团队协作开发以及后续功能扩展。

## 2. 项目目标

- 提供稳定、清晰、易用的手机日程管理能力
- 支持本地离线存储，断网情况下仍可正常使用
- 支持多种提醒方式与免打扰规则
- 支持重复日程和单次例外修改
- 支持 JSON / Excel 数据备份与恢复
- 保证界面交互流畅，适配手机端使用习惯

## 3. 技术选型

- 开发语言：Java
- 开发平台：Android
- 架构模式：MVVM + Repository + Room
- 本地数据库：SQLite（通过 Room 封装）
- 通知方式：本地通知 + AlarmManager / WorkManager
- 并发模型：父线程统一调度 + 子线程执行任务
- UI 风格：Material Design

## 4. 核心功能

### 4.1 日程管理
- 创建日程：标题、描述、开始时间、结束时间、地点、重要程度、标签
- 编辑日程：支持修改所有已创建日程信息
- 删除日程：支持单条删除与批量删除
- 批量操作：支持批量标记完成
- 视图展示：支持日视图、周视图、月视图
- 日期标记：月历中用红点标记有日程的日期
- 重复日程：支持每日、每周、每月、自定义重复
- 例外修改：支持单独修改某一次重复日程

### 4.2 提醒功能
- 提前提醒：支持提前 5 分钟、15 分钟、30 分钟、1 小时或自定义时间
- 重复提醒：未处理日程可按设定间隔重复提醒
- 提醒方式：声音提醒、弹窗提醒
- 免打扰模式：支持设置免打扰时间段
- 优先级穿透：高重要程度日程可穿透免打扰

### 4.3 数据管理
- 本地存储：默认本地保存日程数据
- 导出备份：支持 JSON / Excel 导出
- 导入恢复：支持从备份文件恢复数据
- 数据清理：支持清理已完成或过期日程

### 4.4 界面交互
- 首页：月视图日历 + 当日日程列表
- 详情页：查看、编辑、删除、标记完成
- 添加页：快速录入日程信息
- 设置页：提醒、主题、数据、账号、关于我们
- 主题切换：浅色 / 深色 / 跟随系统
- 手势优化：左滑删除、右滑标记完成

## 6. 并发设计说明

项目采用“父线程统一调度 + 子线程执行任务”的并发模式：

- 主线程：仅负责 UI 渲染与交互响应
- 父线程调度层：负责统一管理任务提交、线程池分配、任务回调
- 子线程执行层：负责数据库读写、提醒计算、重复日程生成、导入导出、数据清理等后台任务

推荐线程池划分：
- `diskIOExecutor`：数据库和文件读写
- `computeExecutor`：时间规则计算和批量处理
- `notificationExecutor`：提醒与通知相关任务
- `mainThreadExecutor`：切回主线程更新 UI

## 7. 开发规范摘要

- 页面层不得直接操作数据库
- 页面层不得直接创建线程
- 所有后台任务必须统一交给调度器处理
- 所有 UI 更新必须回到主线程执行
- 业务逻辑必须按模块拆分，保持职责单一
- 导入导出、提醒、重复规则逻辑必须独立封装
